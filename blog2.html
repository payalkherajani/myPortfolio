Markdown Commands !

Markdown is text-formatting language to style our text and display it !

Some most used commands syntax in Markdown are as follows :-
S.NO Command Description
1 #heading h1 tag
2 **bold-text** Makes text bold
3 *italic-text* Makes text font as Italic
4 ![Logo] (path of image file) Images
5 [title] (https://google.com/) Adds links
6 ` Write code in-between backticks ` Add code.
7 --- Adds Horizontal rule
8 >blockquote-text Add Blockquote
