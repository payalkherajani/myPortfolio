Git Commands !

Git is a version control system which simply track changes made to files.

Git Commands are as follows :-
S.NO Command Description
1 $git --version To check version of git.
2 $git init Initialize your project as git repo.
3 $git status Checking the status of all files in your project i.e. which files are modified or added etc.
4 $git add . Adding all files which are changed.
5 $git add index.html Add a single file(index.html) .
6 $git rm --cached index.html Remove file index.html .
7 $git commit -m 'added-files Commit changes into your repository.
8 $git checkout New-Branch To create and change to New-Branch simultaneously.
